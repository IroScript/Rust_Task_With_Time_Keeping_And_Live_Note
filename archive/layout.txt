## SECTION 8 ‚Äî SESSION TRACKING (Background Logic, No Visible UI)

Runs silently in background via Electron IPC:
- On app load + first rotation start: sends `start-text-run` with current quote text and interval setting
- On every rotation: sends `update-text-stats` with runId, rotation count, elapsed minutes
- On Export Report click: flushes final stats before export (300ms delay), then sends `export-report`
- On window close (beforeunload): sends final `update-text-stats` then `end-text-run`
- Receives: `text-run-started` (gets runId back), `export-success` (shows alert with file path), `export-error` (shows error alert)
